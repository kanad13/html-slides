Minimal Async Fetch Pattern

Error-safe, composable, and easy to test

api.ts 
// Generic fetch wrapper with typed errors
async function fetchData<T>(
  url: string,
  options?: RequestInit
): Promise<{ data: T | null; error: string | null }> {
  try {
    const res = await fetch(url, options);
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    const data = await res.json();
    return { data, error: null };
  } catch (err) {
    return { data: null, error: (err as Error).message };
  }
}
Key points: generic type T keeps it reusable, try/catch prevents unhandled rejections, the return shape is consistent so callers never need to check if data is undefined.
