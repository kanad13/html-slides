Template System v2

Build branded slides by changing tokens, not layouts.

This starter template demonstrates the token-based structure used across the presentation system. Copy it when creating new slide layouts.

Success Warning Information
01

Use tokens

Components consume semantic variables such as brand, surface, text, border, semantic, and chart tokens.

02

Keep layouts stable

A new brand should not require a new layout. Update token values and keep the structure intact.

03

Use colour with intent

Brand colours create identity. Semantic colours communicate meaning. Chart colours support data.
