Custom themes

Inspiration is input, not implementation.

Brand names, screenshots, URLs, mood words, and hex values should be interpreted first, then mapped into the canonical token contract.

1Interpret the sourceIdentify visual intent, audience fit, and theme mode.
2Map token rolesAssign surface, text, brand, semantic, and chart values.
3Check readabilityAdjust derived values when exact source colors do not work.
4Document decisionsRecord source, rationale, and tradeoffs in deck-context.md.
Make the distinction explicit: external references can inspire the token pack, but generated slide components should not copy arbitrary external CSS.
