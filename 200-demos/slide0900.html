Ready to start?

HTML Deck Studio works now. No setup, no dependencies for basic use.

1. Open the viewer

Double-click 100-viewer.html in your file browser or drag it into your browser.

2. Choose a slide folder

Pick any folder with slide0100.html, slide0200.html, etc., and present.

3. Generate with AI (optional)

Give a coding agent the 300-templates/ folder to create new decks from scratch.

4. Export to PDF (optional)

Use 600-tools/export_pdf.py to create production-ready PDFs for sharing or printing.

HTML Deck Studio is modular. Start with the viewer, add generation and export as you need them. Everything is local, everything stays yours.

Optional toolchain

PDF export belongs outside normal viewer use.

The core viewer stays dependency-free. Export can use an explicit Python environment with Playwright and Chromium because it is a separate production task.

One command after setup

python3 600-tools/export_pdf.py 200-demos --out demo.pdf
Position PDF export as an optional toolchain exception. It should never become a requirement for opening or presenting the viewer.
