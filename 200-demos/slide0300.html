Theme system

Tokens define roles before components define appearance.

Custom visual inspiration is useful only after it is translated into stable surface, text, brand, semantic, and chart roles.

Neutral

Surfaces, readable text, dividers, and hierarchy.

Brand

Primary and supporting accents for visual identity.

Semantic

Status colors remain understandable as success, warning, danger, and info.

Chart

Data colors stay distinct on the selected background.
